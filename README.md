# Chat App

A real-time chat application built using React, Node.js, Socket.io, and WebRTC.

## Features

- Real-time messaging
- Voice calls
- Video calls
- Emoji support
- File sharing
- Image sharing
- Message seen ticks
- Edit/Delete messages
- Responsive UI
- Online users
- Typing indicator

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS
- Socket.io Client
- Simple Peer

Backend:
- Node.js
- Express.js
- Socket.io

## Installation

### Clone Repository

```bash
git clone https://github.com/GnaneshwariYeddula/ChatApp.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
